# Thorbit Landing Page

Modern, high-converting landing page for Thorbit - the topical authority measurement platform.

## Features

- ✅ Next.js 14 with App Router
- ✅ TypeScript for type safety
- ✅ Tailwind CSS + Custom design system
- ✅ Fully responsive design
- ✅ Optimized for Vercel deployment
- ✅ SEO-ready with metadata

## Getting Started

### Install dependencies:

```bash
npm install
```

### Run development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.

### Build for production:

```bash
npm run build
npm start
```

## Deploy to Vercel

This project is optimized for Vercel deployment:

1. Push to GitHub
2. Import in Vercel
3. Deploy automatically

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/thorbit-landing)

## Design System

Uses the Mocha Mousse color palette:
- Primary: `#C4704F` (terracotta)
- Accent: `#D9A854` (golden)
- Background: `#FDFBF7` (cream)
- Font: GT Flexa (900 for headlines, 300 for body)
